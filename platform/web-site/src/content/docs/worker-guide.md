## 先完成实名

接单前需完成实名核验：**公安二要素 + 银行卡四要素 + 人脸核身**。实名信息中的身份证号在平台内**信封加密**存储，仅在投保、涉税报送等必要场景解密并强制审计。实名通过后与平台电子签《分包协议（框架）》，并自动开立存管子账户。

## 接单的两条路径

- **报名**：在任务大厅浏览、筛选（地点 / 工种 / 报酬区间 / 匹配我的技能），主动报名，等待企业录用。
- **派单**：企业定向邀约，你在「我的接单」看到带**收入预估**的派单，接受后即与平台电子签《分包工单》、按单投保，任务开始。

## 交付与验收

按任务标准完成并提交交付物。企业验收通过后进入结算。交付 7 天未验收会提醒企业，14 天**自动验收结算**，保护你的报酬。

## 收入与税务

收入页展示**连续接单月份数**、**已累计减除费用**（5000 元 × 月份）与断月重置说明。作为自然人，你的劳务报酬按**累计预扣法**预缴个税——全年多笔小额的预缴税负，通常远低于按次预扣。平台代扣的是预缴税款，最终税负以年度汇算为准。

## 保障与提现

接单即按单投保，配送 / 安装 / 施工等线下作业自动适配**高保额**；出险可在小程序一键报案。报酬经审核后 T+1 提现到绑定的银行卡。

## 个体户专区（B 线）

当月收入达到阈值时，收入页会引导你登记**个体工商户**转入经营所得线：结算不再代扣个税，并解除接单锁定。是否转线由你决定。
