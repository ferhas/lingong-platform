## 争议状态机

当一笔任务发生争议，平台介入仲裁，流程按状态机推进：

1. **协商期（48 小时）**：双方可在线和解、撤回。
2. **平台仲裁**：进入举证期（72 小时），双方提交证据。
3. **裁决**：全额结算 / 部分结算 / 不予结算 / 限期重交付。
4. **执行 → 归档**：裁决执行后留痕归档。

超时由定时任务自动流转，不会卡死。

## 争议锁：先冻结，再裁决

争议一旦创建，立即**冻结该任务的结算**——既不解冻、也不分账，直到裁决产生。这保证了争议期间资金不会被任何一方单方面拿走。

## 按裁决部分结算

裁决为「部分结算」时，平台按裁决金额计酬算税、企业按比例承担承揽价、剩余部分解冻退回，发票按**实际承担额**开具。若任务已结算，争议仅作记录性裁决，不逆向扣划已提现资金。

## 全程留痕

举证、留言、裁决、执行的每一步都进入**时间线留痕**。如对裁决不服，可声明线下升级，平台已备好四流证据包供举证使用。
